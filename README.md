## WorshipSync

Aplicación para planificación y gestión de equipos de adoración.

## Objetivo

Crear una plataforma para iglesias que permita gestionar:
- Canciones
- Setlists
- Servicios
- Equipos de músicos
- Permisos de usuarios
- Modo en vivo para cultos

## Tecnología

Frontend: Next.js  
Backend: NestJS  
Base de datos: PostgreSQL WorshipSync
App para canciones y alabanzas
