# WorshipSync Architecture

## Objetivo
Plataforma para gestión de equipos de adoración e iglesias.

## Arquitectura
Frontend: Next.js + React  
Backend: NestJS + TypeScript  
Database: PostgreSQL + Prisma

## Módulos principales
- Autenticación
- Iglesias
- Usuarios y permisos
- Canciones
- Servicios
- Setlists
- Modo en vivo
- Notificaciones
